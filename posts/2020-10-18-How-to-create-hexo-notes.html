如何创建hexo的笔记呢？

2020-10-18

这是一篇很简单的创建hexo的笔记，笔者会尽快能的写详细。

如何你还没有 hexo，并且不会的，请访问ismdeep.com,这位大

给你很好帮助。

第一步 打开我的 blog，点击 hexo 的命令窗口，

输入命令 hexo new （空格）“2020-XX-XX-(连上这个文档标题)” 换行

请注意，一定要写上时间

当输入 hexo new “XXXX”，下一行出现了INFO Created…表示创建成功。

输入下一个命令 hexo server （表示连上服务器），下一行出现了

INFO Start Processing 表示成功了，就可以打开谷歌浏览器

http://locahost:4000 可以一边编辑一边看你的博客上有编辑那些笔记

第二步，就算编辑的过程

打开blog–>source–>-posts 找到新建的markdown

需要在 title 处改掉时间，只需博客文章的标题

还有在 tags 处写标签 是在 [] 里面写，要逗号，空格隔开

好比[study，think]

然后，就编辑你的博客吧

请注意，一点要Ctrl +s 保存

最后，回到hexo的命令窗口

确定没有问题了，

就Ctrl+c 结束 hexo server

输入 hexo generate （生成）

也可以 hexo g

输入hexo deploy （部署）

也可以 hexo d

笔者推荐 hexo generate，hexo deploy

然后，就是要填你的用户名，和密码

等上传几分钟，你的博客就完成了

笔者还不会上传图片，看起来干巴巴，笔者会继续努力。
