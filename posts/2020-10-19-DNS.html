DNS的配置

2020-10-19

DNS

今天，笔者将讲讲DNS如何配置。

DNS，笔者相信百度，谷歌，火狐，可以给你很好的解析

先确定我们安装的虚拟机是否可以上网

如何配置网络呢？，很多博客平台都会有这方面的知识。

配置DNS

安装BIND yum install bind -y

需要记住服务器使用的ip

输入命令vi /etc/named.conf

是在named.conf 文件里面改，不需要改其他的地方

设置zone(域)及其他相关文件

vi /etc/named.rfc1912.zones

“kgc.com.” 这是可以改其他的，com. 最好还是保留

编写ZONE文件

vi /var/named/named/kgc.com.zone

$TTL 1D
IN SOA kgc.com. admin.kgc.com.(
0 ; serial
1D ; refresh
1H ; retry
1w ; expire
3H ) ; minimum
NS kgc.com.
A 192.168.15.100 (自己的ip地址)
www IN A 192.168.15.101
ftp IN A 192.168.15.102
stmp IN CNAME WWW

设置本地域名解析服务器地址

vi /etc/resolv.conf

启动域名解析服务器

systemctl stop firewalld setenforce 0 systemctl start named

验证的结果

输入命令

ping www.kgc.com

会出现 www.kgc.com(192.168.15.100)…

ping ftp.kgc.com

会出现 www.kgc.com(192.168.15.100)…

ping stmp.kgc.com

会出现 www.kgc.com(192.168.15.100)…
