linux 学习与运用

2020-10-22

笔者花了一下午的时间，去写了这篇文章。依然对很多命令都不熟悉，其中，也有一些题是在网上搜的。

文件和目录类的命令

利用cp命令复制系统文件/etc/ profile到当前目录下

cp /etc/profile /root/test(当前目录)

什么是当前目录（如，在home，那一定是home目录下，cd test，那当前目录一定是test）

cp /etc/profile newfile是/etc/profile 复制到newfile

复制文件profile 到一个新文件profile.bak 作为备份

cp profile profile.bak

用grep命令在profile文件对关键字then进行查询

grep then profile

用less命令分屏查看文件profile的内容，

less profile

注意比较练习less命令的各个子命令

- b 移动上一页

- q 退出

- p

less -p then profile

给文件profile创建一个符号链接Insprofile

ln -s profile lnsprofile

和硬链接Inhprofile

ln profile lnhprofile

以长格式显示文件 profile lnsprofile lnhprofile ,主要比较不同

ll

如何回到上层目录

cd .. . 代表一层目录

tar 的子命令

- c 生成档案文件

- v 列出归档，解档的详细过程

- f 指定档案文件名称

- r 将文件追加到档案文件末尾

- z 以gzip格式压缩或解压缩文件

- j 以bzip2 格式压缩或解压缩文件

- d 比较文档与当前目录的文件

- x 解开档案文件

用tar命令把test目录打包

tar -zcf test.tar teat

用gzip命令把打好的包进行压缩

gzip test.tar

把文件test.tar.gz 改名为backup .tar.gz

mv test.tar.gz backup.tar.gz

把文件backup.tar.gz 移动到test目录下

mv backup.tar.gz /test

把文件backup.tar.gz 解包

tar - x backup.tar.gz

复制test目录为testbak目录作为备份

cp test testbak

查看root 用户自己的主目录下的所有文件名为profile

find -mane profile

删除test 子目录下的所有文件

rm -f test/test*

系统信息命令

利用data命令显示系统当前的时间，并修改系统当前时间

data

显示当前登录到系统的用户状态

长时间在用root权限，忘记了自己的用户是什么了

利用free命令显示内存的使用情况

利用df命令显示系统硬盘分区及使用情况

显示当前目录下的各级子目录的硬盘占用情况

du -a

进程管理类命令

  • 使用ps命令查看和控制进程

    显示本用户的进程

    ps -u

    显示所有用户的进程

    ps -u root

在后台运行cat命令

查看进程cat

ctrl + D

杀死进程cat

再次查看进程cat，看其是否被杀死

  • 使用top命令查看和控制进程

    用top命令动态显示当前的进程

    只显示用户user01 的进程

    利用k键杀死指定进程号的进程

  • 挂起和恢复进程

    执行命令cat

    按Ctrl+Z 组合键挂起进程cat

    输入jobs 命令，查看作业

    输入bg，把cat切换到后台执行

    输入fg，把cat切换到前台执行

    按Ctrl+C 组合键结束进程cat
